# Creative Pipeline — AI Meta Ad Generator

Complete AI-powered pipeline from research documents to 40 generated Meta ad creatives with feedback loop.

---

## Quick Start (5 minutes)

### 1. Install dependencies
```bash
cd backend && npm install
cd ../frontend && npm install
```

### 2. Configure API keys
```bash
cd backend
cp .env.example .env
# Edit .env and add your keys:
# CLAUDE_API_KEY=sk-ant-api03-...
# GEMINI_API_KEY=AIzaSy...
```

Get Claude key: https://console.anthropic.com
Get Gemini key: https://aistudio.google.com/apikey

### 3. Start the servers
```bash
# Terminal 1 — Backend
cd backend && npm start

# Terminal 2 — Frontend  
cd frontend && npm start
```

Open http://localhost:3000

## The Flow

```
Typeform → Make.com → 5 Research Docs in Google Drive
                              ↓
                    Download from Drive
                              ↓
                    Upload in Dashboard (Upload tab)
                              ↓
                    Run Pipeline (Run tab)
                      ├── Research: Claude reads all docs → master context
                      ├── Strategy: 20 formats × 2 angles = 40 briefs
                      ├── Creative: Claude prompt → Nano Banana image × 40
                      └── Review: AI scores all 40, ranks top 10
                              ↓
                    Download top 10 → Upload to Meta manually
                              ↓
                    Run ads 5-7 days
                              ↓
                    Export Meta CSV → Upload in Feedback tab
                              ↓
                    Claude analyzes + generates improved iterations
                              ↓
                    Repeat weekly
```

## Documents to Upload

Generated by your Make.com automation from Typeform:
- `02-Market-Research` — market landscape, competitors, buying psychology
- `03-Avatar-Sheet` — persona, pain points, demographics, exact quotes
- `04-Offer-Brief` — USP, mechanism, proof points, ad angles
- `05-Necessary-Beliefs` — objections, belief sequence
- `Customer Voice Research` — verbatim Reddit quotes, tone of voice
- Optional: `06-Master-Summary`, brandbook PDF

## Meta CSV Format

Export from Meta Ads Manager with these columns:
```
ad_name, impressions, reach, clicks, ctr, cpc, cpm, spend,
conversions, cpa, conversion_rate, thumb_stop_rate, frequency
```

**Critical:** The `ad_name` column must exactly match the file names: `FORMAT-01-VERSION-A`

Download template from the dashboard: Feedback tab → Download Template

---

## How the Feedback Loop Remembers Previous Work

After the initial run, the system saves:
- `master_context.json` — full client research synthesis
- `content_briefs.json` — all 40 briefs with hooks, headlines, body copy
- `creative_manifest.json` — all prompts and visual directions

When you upload Meta CSV data a week later, Claude reads all of this saved data alongside the performance metrics. It knows exactly what headline caused a low CTR, what hook generated high thumb-stop rate, what visual direction worked — and makes specific, informed improvements rather than starting from scratch.
